# CloudQA Automation Practice Form — Selenium C# Tests

**What**: Automated tests for the AutomationPracticeForm page (First Name, Gender, Country) using Selenium + C#.  
**How to run**:
  - Clone repo  
  - Open solution in Visual Studio  
  - Restore NuGet packages  
  - Build solution  
  - Run **Test → Test Explorer → Run All**  
**Browser used**: Chrome  
**Notes**: Locators are based on visible label text + XPath, so tests are robust to changes in element IDs or page layout.
